# 🎯 ROLE & OBJECTIVE
Bertindaklah sebagai Senior Full-Stack Developer dan UI/UX Designer. 
Saya ingin menambahkan fitur baru **"SNBT Progress Tracker"** ke dalam aplikasi web saya.

**Tujuan Fitur:** 
Memungkinkan pengguna untuk melihat daftar materi UTBK (dari file `silabus.js`), mempelajari materinya, dan menandai (checklist) materi yang sudah dikuasai. Sistem harus menyimpan progres ini secara persisten per user.

# 🛠️ TECH STACK & CONSTRAINTS
1. **Data Source:** Daftar materi diambil dari file lokal `silabus.js` (struktur: Array of Objects dengan properti seperti `id`, `subject`, `topic`, `subtopic`).
2. **Database:** Firebase Firestore (NoSQL).
3. **Design Style:** Minimalis, Elegan, Simpel, Premium (Clean lines, ample whitespace, subtle shadows, professional typography).
4. **UX Principle:** User harus bisa melihat progres global (% selesai) dan detail per bab.
