## State Management

### Component State

```javascript
// Admin Status
isAdmin: boolean
  в”ңв”ҖвҶ’ true: User has admin role
  в””в”ҖвҶ’ false: User is regular user

// Delete Confirmation
showDeleteConfirm: null | string
  в”ңв”ҖвҶ’ null: Dialog not shown
  в””в”ҖвҶ’ setId: Dialog shown for this set

// Data States
myQuestions: QuestionSet[]
publicQuestions: QuestionSet[]
  в””в”ҖвҶ’ Updated after deletion

// Loading States
loading: boolean
  в””в”ҖвҶ’ true: Data loading
```
